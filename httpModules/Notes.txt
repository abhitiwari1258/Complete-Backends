1. What is HTTP?

HTTP (HyperText Transfer Protocol) is the protocol used for communication between: Client (Browser) and Server (Node.js)

Importing HTTP Module
const http = require("http");


HTTP Status Codes

Success
200 OK
201 Created
204 No Content

Client Errors
400 Bad Request
401 Unauthorized
403 Forbidden
404 Not Found

Server Errors
500 Internal Server Error
503 Service Unavailable
